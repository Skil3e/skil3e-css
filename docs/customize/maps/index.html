Maps

Maps are used to generate themes and utility classes.

Add to map

You can add values to a map with the map-merge function. First you @use the maps file and then you overwrite the map on the file that you want to use,

In the example bellow we will add a color to the themes.

//First import the sass function
@use "sass:color";

//Then import the maps from the library
@use "~skil3e-css/src/maps";

//Finally to merge the values to the map on the file you want.
@use "~skil3e-css/src/basic/themes" with (
    $themeLight: map-merge(maps.$themeLight, ("bg2": #eeeeee)),
    $themeDark: map-merge(maps.$themeDark, ("bg2": #111111))    
);
